# Maestría en Inteligencia Artificial Aplicada – UIDE

Repositorio con ejercicios prácticos desarrollados en la asignatura Procesamiento de Imágenes y Señales de la Maestría en Inteligencia Artificial Aplicada de la UIDE.

Los notebooks incluyen la aplicación de técnicas básicas y avanzadas de procesamiento digital, tales como manipulación de imágenes, filtrado, detección de bordes, análisis de características y captura desde cámara, utilizando Python y librerías vistas en clase.
